Método RK4

Este método perimite el estudio de sistemas dinamicos, en otras palabras, sistemas que cambian su estado con el tiempo.

Los sistemas dinámicos intentan resolver la trayectoria temporal de alguna cantidad física como función de algún generador dinámico.

En algunos casos, podemos modelar la dinámica mediante la siguiente relación:

\[\begin{equation} \frac{dy}{dt} = f(t, y) \end{equation}\]

sujeta a la condición inicial:

\[\begin{equation} y(_{0}) = y_{0} \end{equation}\]

  • Con \(y\) un estado del sistema.

  • Con \(t\) una variable temporal.

Este problema es conocido tambien como problema de condición inicial.
